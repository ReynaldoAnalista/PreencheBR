Importar regras via JSON

Cole um array JSON com regras personalizadas. As regras importadas são mescladas às existentes. Ver exemplo 

[
  { "label": "Tipo de representação", "valor": "Procurador", "tipoCampo": "select" },
  { "label": "Status", "valor": "Ativo", "tipoCampo": "qualquer" }
]

Nova regra

Define o que preencher quando o label/contexto de um campo contiver o texto informado. Funciona em inputs, textareas e selects (para selects, o valor deve corresponder a uma das opções).

Trecho do label, placeholder ou nome do campo no formulário.
Para selects: texto da opção (correspondência parcial). Para checkboxes: true para marcar, false para desmarcar.

Regras salvas
